# Summarizer

Summarizer is a browser plug-in based on the [wxt framework](https://github.com/wxt-dev/wxt) that I learned and developed myself.

![](./screenshots/screenshot.png)

## Installation

```bash
# or npm or yarn
pnpm install
```

## Run
```bash
pnpm dev
```
